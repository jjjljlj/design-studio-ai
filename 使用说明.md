# Design Studio AI 使用说明

这是一个面向服装客户的设计开发应用原型，当前已经可以本地运行。

## 打开方式

双击：

```text
start-app.bat
```

然后在浏览器打开：

```text
http://localhost:5173
```

客户演示页：

```text
http://localhost:5173/customer.html
```

## 当前功能

- 客户需求录入
- 款式开发方案生成
- 花型设计方向生成
- 产品图提示词生成
- 数码矢量图提示词生成
- 英文标题、五点描述、关键词生成
- 样衣 / 参考图本地预览
- 无 API Key 时自动进入演示模式

## 接入 ChatGPT / OpenAI 模型

在项目根目录新建 `.env` 文件，填入：

```text
OPENAI_API_KEY=你的 OpenAI API Key
OPENAI_TEXT_MODEL=gpt-5.5
OPENAI_IMAGE_MODEL=gpt-image-2
PORT=5173
```

保存后重新双击 `start-app.bat`。

## 后续可以继续开发的模块

- 客户登录和项目列表
- 设计订单、报价、交付状态
- 样衣图片自动识别
- 生成图片保存到项目文件夹
- 花型矢量稿下载
- 客户确认 / 修改意见流程
- 后台审核，防止侵权、仿牌和虚假宣传
